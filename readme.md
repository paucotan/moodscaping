# Moodscaping

Moodscaping is a minimalist web app that generates inspirational moods—pairs of random quotes and images—to evoke reflection and inspiration with a single click.

## Features
- Generate a random mood with a quote and matching image
- Fetches data dynamically from:
  - [ZenQuotes API](https://zenquotes.io/) for random quotes
- Smooth transitions for an immersive experience

## Tech Stack
- HTML5 for structure
- CSS3 for styling
- Vanilla JavaScript (ES6) for logic and API integration

## Usage
Click the “Random” button to generate a new mood with a random quote and image. The background and text update smoothly to create a calming atmosphere.

## Goal
Moodscaping serves as a creative exercise in front-end development, API interaction, and crafting elegant user experiences.
